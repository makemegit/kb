# __Firefox Tweaks__

---

## ___add .lan to firefox___

```bash
about:config
browser.fixup.domainsuffixwhitelist.lan -> add -> true
```

## ___How to always show full address or URL in Firefox (show http, show https)___

```bash
about:config
browser.urlbar.trimURLs -> change to false
```
